#### 软件架构

```
dd-cat-cloud
├── cloud-client -- 客户端模块
├    └── client-nacos -- nacos注册中心
├── cloud-common -- 公共模块
├── cloud-drools -- 规则引擎
├── cloud-sharding -- 分库分表 读写分离
├── cloud-transaction -- 事务
├    └── cloud-transaction-seata -- 分布式事务
├── cloud-workflow -- 工作流
└──  └── workflow-flowable -- flowable工作流
```
